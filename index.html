    functions

    palette

    @function palette($color, $shade: 500) { ... }

    Description

    Funzione in grado di attraversare la mappa dei colori utilizzati nel Material Design

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $color

    Il nome del colore definito nella mappatura del Material Design

    		String—none
    $shade

    Permette di definire il grado del colore che vogliamo utilizzare

    		String/int500

    Returns

    Hex —

    Il valore esadecimale del colore utilizzato

    Example

    Uso base per creare un colore di sfondo

    .ele{
  color: palette(Light Blue, 700);
}

    Requires

    • [variable] palette

    mixins

    bshadow

    @mixin bshadow($case) { ... }

    Description

    Con questo mixin creo le ombre rispettando i valori del Material Design

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $case

    Permette di definire il tipo di ombra material da utilizzare

    		Int—none

    Example

    Uso senza parametri

    .elemento-con-ombra{
  @include bshadow();
}

    Compilato senza parametri

    .elemento-con-ombra{
  -moz-box-shadow: rgba(0, 0, 0, 0.12) 0 1px 3px, rgba(0, 0, 0, 0.24) 0 1px 2px;
  -webkit-box-shadow: rgba(0, 0, 0, 0.12) 0 1px 3px, rgba(0, 0, 0, 0.24) 0 1px 2px;
  box-shadow: rgba(0, 0, 0, 0.12) 0 1px 3px, rgba(0, 0, 0, 0.24) 0 1px 2px;
}
